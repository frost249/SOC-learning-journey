## Challenge Investigation (Snapped Phish-ing Line)

### Scenario

A phishing campaign targeted multiple employees. A malicious email sample and attachment were provided for investigation inside a controlled VM.

---

### Investigation Summary

**Initial Analysis**
- Reviewed phishing email in Thunderbird  
- Identified malicious HTML attachment  
- Extracted redirection URL from source code  

**Infrastructure Analysis**
- Resolved domain `kennroads.buzz`  
- Manually enumerated `/data` directory  
- Discovered exposed archive: `/data/Update365.zip`  

**Phishing Kit Analysis**
- Extracted phishing kit contents  
- Reviewed `submit.php` credential harvesting script  
- Identified credential exfiltration via PHP `mail()` function  

**Threat Actor Artifacts Identified**
- Primary exfiltration email: `m3npat@yandex.com`  
- Additional email: `jamestanner2299@gmail.com`  

**Flag Discovery**
- Located `/data/flag.txt`  
- Flag was Base64 encoded and reversed  
- Decoded using:

```bash
echo "<encoded_flag>" | base64 -d | rev
```
